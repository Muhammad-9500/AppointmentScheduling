# 📅 Appointment Scheduling System

An Appointment Scheduling System built with **ASP.NET Core MVC**, **Entity Framework Core**, and **SQL Server** that enables users to schedule, manage, and track appointments through an intuitive web interface.

This project demonstrates the development of a business management application using the MVC architecture while implementing CRUD operations, authentication, and database-driven workflows.

---

# ✨ Features

- Appointment Scheduling
- Create Appointments
- Update Appointments
- Cancel Appointments
- Appointment History
- User Authentication
- CRUD Operations
- SQL Server Database Integration

# 🚀 Getting Started

## Prerequisites

- .NET SDK
- SQL Server
- Visual Studio 2022

### Clone the repository

```bash
git clone https://github.com/Muhammad-9500/AppointmentScheduling.git
```

### Configure the database

Update the SQL Server connection string inside:

```
appsettings.json
```

Run Entity Framework migrations:

```bash
dotnet ef database update
```

Run the application:

```bash
dotnet run
```

or simply press **F5** in Visual Studio.
